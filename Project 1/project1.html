Welcome to REKT 'EM!

This game is about trying to lose your hand the fastest! However, there are ways to make this goal harder. Read the rules for more information!

Rules

  • The only way to put down a card is to have a number higher than the "Card to Beat"
  • The only way you can put a lower card is to either put down a King, Queen, Jack, or Ace
  • If you only have one card left in your hand, you will need to yell "REKT 'EM" beforehand or else you will draw one extra card
  • If someone puts down a King, the opponent must draw 3 cards
  • If someone puts down a Queen, the opponent must draw 2 cards
  • If someone puts down a Jack, the opponent must draw 1 card
  • If someone puts down an Ace, the opponent must draw 5 cards
